# Verdant Fork Website

## Project Overview

I created a multi-page website for a plant-based restaurant called Verdant Fork. The goal of this project was to take my high-fidelity designs from my User Experience Design course and turn them into a fully functional website using HTML, CSS, and Bootstrap.

The website includes four main pages:
- Home
- Menu
- Careers
- About

Each page was designed to be consistent, easy to navigate, and responsive across different screen sizes.

---

## My Process

I started by using my Figma designs as a guide. I built the HTML structure for each page first, focusing on layout and content placement.

After that, I added CSS to match my design, including colors, spacing, and typography. I used Flexbox and CSS Grid to organize sections like the menu layout and categories.

I then added Bootstrap to support layout structure and responsiveness. I used Bootstrap for containers, spacing, and grid alignment, while customizing everything with my own CSS to match my design.

Finally, I tested the website using browser inspect tools to make sure it worked on desktop, tablet, and mobile.

---

## Challenges I Faced

One challenge was keeping the layout consistent across different screen sizes. Some sections, like the menu and navigation, would break or misalign on smaller screens. I solved this by using media queries and adjusting layout behavior.

Another challenge was combining Bootstrap with my custom CSS. Sometimes Bootstrap styles would override my design. I fixed this by using more specific CSS selectors and organizing my styles better.

I also had to adjust positioning multiple times, especially for sections like the menu and buttons, to match my original design.

---

## What I Learned

I learned how to build a complete multi-page website from scratch and keep it consistent across pages.

I improved my understanding of:
- Flexbox and CSS Grid
- Responsive design using media queries
- How Bootstrap works and how to customize it

I also learned how small changes in CSS can affect the entire layout and how to debug those issues effectively.

## Responsive Design

I used media queries to adapt the layout for:
- Desktop
- Tablet
- Mobile

On smaller screens, layouts stack vertically, grid columns reduce, and text and images resize to maintain readability.
